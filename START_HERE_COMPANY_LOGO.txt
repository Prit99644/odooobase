## Logo Recommendations

### What Works Well âœ“
- Company logo/brand mark
- Company initials with background
- Industry-specific icon
- Company symbol/mascot
- Text logo with company name

### Specifications
- Format: PNG (best) or JPG
- Size: 100 Ã— 50 pixels (recommended)
- Max Size: 2 MB (enforced)
- Display: 40px height, auto-width

### What to Avoid âœ—
- Photos (use logo instead)
- Very large files (> 1 MB)
- Very dark colors (hard to see on purple navbar)
- Text that's too small to read

## Troubleshooting

### Logo Not Showing?

**1. Check directory exists:**
```
c:\xampp\htdocs\hrms\assets\images\logos\
Should exist and be empty initially
```

**2. Check logo uploaded:**
```
After registration, check folder has files like:
1704268800_CompanyName.png
```

**3. Check database:**
```
phpMyAdmin â†’ hrms_db â†’ users table
company_logo column should have path:
assets/images/logos/1704268800_CompanyName.png
```

**4. Clear cache:**
```
Browser: Ctrl+Shift+Delete (clear cache)
Reload: F5 or Ctrl+R
```

### Upload Fails?

**File too large:** Max 2MB (compress if needed)
**Wrong format:** Use PNG, JPG, or GIF only
**Directory issue:** Check assets/images/logos/ is writable
